This is where it all begins...
To start, click the and download API packs that you're interested in. A short sample for the scala pack to get you started:
scala Future andThen
Search queries consist of up to three components separated by spaces:
  1. Pack Name
  2. Path
  3. Member
The first is the only required component and identifies the documention pack that you are interested in. The second part matches on the path and the third one on the entity that your are looking for. The above example matches a method in Scala's API, where the full path is:
scala.concurrent.Future
Give it a try!

All three parts are regular expressions:
sc utu.+ ^andThen$
This yields at least the same results as the first sample. A pattern is automatically case insensitive if it contains no upper case letter. For example the following matches at least the same as the previous sample:
sc utu.+ ^andthen$
You can use the following keys for navigation:
  • /This moves focus to the search field unless focus is in the documentation iframe.
  • shift + →This selects the next search result to the right, if possible.
  • shift + ←This selects the next search result to the right, if possible.
The documentation frame is the iframe element that is displaying this readme at the moment. This iframe is used to isolate the docs while the navigation and search remains available on top as a header.
